# 2. How Do You Declare Variables in JavaScript?
let & const are used to declare variables

## Example

# 3. Rules for Naming Variables

 - Names cannot start with numbers 
 - Names cannot what so ever start with with designated Javascript keywords
 - Spacing is not allowed while naming variables
 - Variables are case sensitive so being keen is recommended
 - Only symbols allowed are for example $
#  4. Variable Naming Styles
## Camel case
This style of naming variables consists of the first name starting with a Lowercase while the second name starting with an Uppercase.
### Example
 let studentGrade = "A";
let topScore  = 50;
## Pascal case
This is a style of naming variables consists of  both the first & second word starting out with an upper case.
### Example
let StudentGrade = "A";
let Topscore =50;
##  Snake Case
A snake case is a type of style used in naming variables similar to camel case but in snake case the words are not together instead they are separated with the underscore.
### Example
let student_Grade = "A";
let top_Score =50;
