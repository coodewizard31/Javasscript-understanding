#  4. Variable Naming Styles
## Camel case
This style of naming variables consists of the first name starting with a Lowercase while the second name starting with an Uppercase.
### Example
 let studentGrade = "A";
let topScore  = 50;
## Pascal case
This is a style of naming variables consists of  both the first & second word starting out with an upper case.
### Example
let StudentGrade = "A";
let Topscore =50;
##  Snake Case
A snake case is a type of style used in naming variables ,in snake case the words are not together instead they are separated with the underscore & the starting letters are all lowercase.
### Example
let student_Grade = "A";
let top_Score =50;
